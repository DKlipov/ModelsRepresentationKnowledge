Educational application
For start in telegram, create file token.sec with bot token, update proxy settings and run bot.py
For work in console run console.py
Aviable commands : /help for help; /parse for change mode to parse; /syn for change mode to syn.
In parse mode you need print any string for parsing it.
In syn (synthesis) mode you aviable next templates:
существительное СУ gender number case
прилагательное ПП gender number case
глагол ГЛ gender number
числительное ЧИ case
